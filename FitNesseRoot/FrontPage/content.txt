!1 Welcome to [[FitNesse][FitNesse.FitNesse]]!
!3 ''The fully integrated stand-alone acceptance testing framework and wiki.''

With [[Cucumber][!-CucumberTestSystem-!]] support!

Read, edit and run [[Cucumber][!-CucumberTestSystem-!]] feature files from within FitNesse.

To get started:

 * Pages adhere to the Cucumber feature syntax (Gherkin)
 * Configure the Cucumber test system with FitNesse
 * Define variable ''cucumber.glue'' and let ot point to the class path where the step definitions can be found
